## Контрольные вопросы

1. Intent — механизм обмена сообщениями для запуска компонентов. Явный: указывает конкретный класс — `new Intent(this, TargetActivity.class)`. Неявный: описывает действие — `new Intent(Intent.ACTION_VIEW, Uri.parse("http://..."))`.

2. Передача данных: через `putExtra()` (примитивы, строки, сериализуемые объекты). Ограничения — передаваемые объекты должны реализовать `Serializable` или `Parcelable`.

3. startActivity() — запуск без ожидания результата. startActivityForResult() — запуск с ожиданием результата, обрабатываемым в `onActivityResult()`.

4. setResult() — устанавливает результат перед закрытием Activity. finish() — закрывает Activity. Вместе используются для возврата данных в вызывающую Activity.

5. Без регистрации в манифесте Activity вызовет `ActivityNotFoundException` и краш приложения.

6. Жизненный цикл: при переходе: `onPause()` Main → `onCreate()` → `onStart()` → `onResume()` Settings. При возврате: `onPause()` Settings → `onActivityResult()` → `onResume()` Main.

7. requestCode — идентификатор запроса, позволяет различать источники возврата в `onActivityResult()` через `switch(requestCode)`.
